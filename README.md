# Express Mentoring 용 레포입니다.

- 중간에 k8s, 도커 관련 파일은 무시하셔도 됩니다... (.yaml이랑 express 폴더는 무시하셔도됩니다)

### 프로젝트 초기설정

- git clone

```bash
git clone https://github.com/8471919/express-k8s.git
```

- 터미널에서 clone 받은 폴더로 이동 후,

```bash
# 의존성 패키지 설치
npm install
```

- 서버 시작

```bash
npm run start

# 혹은
npm run start:dev
```
